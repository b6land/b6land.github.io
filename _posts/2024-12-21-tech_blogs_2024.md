今天想跟大家分享一下我今年常看的幾個部落格 (而且作者還有在更新！)，還有我對它們的主觀看法。

### C# 和 SQL Server 相關主題

- [黑暗執行緒](https://blog.darkthread.net/)：作者黑大的文筆風趣，在技術文章裡面風格獨樹一幟，教學淺顯易懂，常常在找 ASP.NET 的問題時被作者的文章拯救。
- [The Will Will Web](https://blog.miniasp.com/)：作者保哥耕耘 .NET 領域已久，近年接觸不少生成式 AI 的服務，讓自己也開始接觸 ChatGPT、Codeium 以外的工具。
- [m@rcus 學習筆記](https://marcus116.blogspot.com/)：比較常更新 Facebook，分享許多 .NET 相關的新知。
- [安德魯的部落格](https://columns.chicken-house.net/)：作者的每篇文章都探討的十分深入，例如架構的設計，希望能在研讀後設計更佳的 API 架構。
- [Huan-Lin 學習筆記](https://www.huanlintalk.com/)：C# 執行緒、非同步程式設計文章的熱門作者，寫到非同步程式就會上去回顧。

### 其它主題

- [軟工現場觀察](https://devindeving.blogspot.com/)：主題為 PHP 語言，推薦作者對於職涯規劃、書單推薦的文章。
- [Huli's blog](https://blog.huli.tw/)：雖然部落格以資安比賽和漏洞為主，但是 Facebook 專頁其實有許多比較貼近生活的資安文章。
- [Code and Me](https://blog.kyomind.tw/)：主題為 Python 語言，作者有很多對於生活方式、筆記系統、軟體開發方式的思考，因此推薦「軟體開發」和「心得」類別的文章。
- [弦而時習之](https://blog.aotoki.me/)：雖然主題為 RoR 語言，推薦與職涯 Tag 相關的文章。
- [嫁給 RD 的 UI Designer](https://blog.akanelee.me/)：雖然職務上不太會碰到 UI，但還是要知道一些版面設計的概念，另外也推薦生活、成長類別的文章。
