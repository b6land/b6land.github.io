本篇介紹 Visual Studio Code (VSCode) 中，怎麼尋找與取代新行。

### 尋找與取代新行

- 在搜尋框中，按下 Ctrl + Enter，就可以找到換行。
- 如果想要把特定文字用換行符號取代，在搜尋欄右方按下 Regex 的按鈕，取代欄輸入 `\n` 即可。
